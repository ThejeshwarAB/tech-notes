# Git Commands

This file is a lightweight reference for commonly used Git commands that I've learnt.  

## Inspecting state
- git status
- git log

## Saving work
- git add
- git commit

## Branching and navigation
- git branch
- git checkout

## Undoing changes
- git reset
- git revert

## Syncing with remotes
- git remote
- git fetch
- git pull
- git push
